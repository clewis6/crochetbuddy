# CrochetBuddy Images Folder

This folder contains all images for the CrochetBuddy website.

## Image Guidelines

### Required Images
- placeholder-1.jpg through placeholder-6.jpg (blog card images)
- bunny-main.jpg (featured blog post image)
- Logo/favicon images

### Recommended Sizes
- Blog card images: 600x400px
- Featured post images: 1200x800px
- Category hero images: 800x600px
- Ad placeholders: Standard IAB sizes (728x90, 300x250, 300x600)

### Optimization Tips
- Use WebP format for better compression
- Optimize images to under 200KB each
- Use descriptive alt text for accessibility
- Consider lazy loading for better performance

### Stock Photo Resources (Free)
- Unsplash.com
- Pexels.com
- Pixabay.com

Search terms: "crochet", "yarn", "amigurumi", "knitting", "crafts", "handmade"
